# Blobscape

A groovy 70s-inspired virtual lava lamp built with Next.js 14.

## Features

- **Retro 70s Aesthetic** - Warm tones, wood grain textures, and funky typography (Righteous + Pacifico fonts)
- **Interactive Color Wheel** - Pick your vibe by clicking anywhere on the color wheel
- **Hypnotic Lava Lamp** - Smooth, blobby shapes that float and wobble organically
- **Analogous Color Generation** - Your selected color spawns complementary hues for visual depth
- **Ambient Color Shift** - Subtle hue changes over time for a meditative experience
- **Fullscreen Mode** - Press spacebar to toggle fullscreen for maximum chill

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- CSS Animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and get groovy.

## Controls

- **Click** the color wheel to select your lamp color
- **Click** anywhere on the lamp to return to color picker
- **Spacebar** toggles fullscreen mode
